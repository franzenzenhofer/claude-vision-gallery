- **3D Generation**: Blender 4.4.3 + Python API
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Fonts**: Space Grotesk, Inter
- **Rendering**: Cycles renderer with 64-256 samples
- **Testing**: Custom Python test suite + Playwright
